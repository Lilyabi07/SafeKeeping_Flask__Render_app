## Overview

This is a Flask-based IoT home security and environmental monitoring application designed to be deployed on Render. The application integrates with various IoT sensors and actuators, stores data in both local SQLite and cloud PostgreSQL (Neon) databases, and provides a web interface for monitoring and control.
